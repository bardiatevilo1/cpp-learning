# C++ Learning

A collection of C++ projects built while learning the language.

## Projects
- **01-calculator** — basic arithmetic CLI, input validation, while loop, hashmaps, lambdas.
- **02-unit-conversion** - converting units of distance, time, mass. 
- **03-stat-calculator** - bubble sort, more input validation.
- **04-prime-sieve** - Implements the Sieve of Eratosthenes to efficiently find all prime numbers up to a user-defined limit.
- **Matrix Addition & Multiplication**: Implements matrix addition and multiplication using 2D arrays and nested loops. Reading and writing to txt files with ifstream and ofstream using fstream.
