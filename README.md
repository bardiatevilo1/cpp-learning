# C++ Learning

A collection of C++ projects built while learning the language.

## Projects

### Fundamentals
- **01-calculator** — basic arithmetic CLI, input validation, while loop, hashmaps, lambdas.
- **02-unit-conversion** - converting units of distance, time, mass.
- **03-stat-calculator** - bubble sort, more input validation.
- **04-prime-sieve** - Implements the Sieve of Eratosthenes to efficiently find all prime numbers up to a user-defined limit.
- **05-matrix-operations**: Implements matrix addition and multiplication using 2D arrays and nested loops. Reading and writing to txt files with ifstream and ofstream using fstream.

### OOP and Data Structures
- **06-vector3D** - Operator-overloaded 2D/3D vector classes supporting arithmetic, dot/cross products, magnitude, and normalisation.
- **07-complex-nums** - Operator-overloaded complex number class with arithmetic, modulus, argument, conjugate, and Euler's formula.
